The portal-kernel (com.liferay.portal.kernel) module API reference documentation describes packages and classes for you to use as you develop applications on the Liferay platform.

Please note that the learning paths and tutorials in the developer section of the Liferay Developer Network provide the best resources to guide you in consuming Liferay services and customizing Liferay. In addition to this, the Developer Network's community section provides community resources and explains how you can participate in the Liferay Community.
